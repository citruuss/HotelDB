# Hotel Management Database System

## Описание проекта
База данных для автоматизации управления гостиницей. Система предназначена для учета постояльцев, управления бронированием номеров, организации дополнительных услуг и автоматизации финансовых операций.

### Цель проекта
Разработка полнофункциональной базы данных для гостиницы с реализацией бизнес-логики на уровне СУБД (триггеры, хранимые процедуры, функции) для обеспечения целостности данных и автоматизации основных операционных процессов.

## Функциональные возможности

- Ведение списка постояльцев с историей проживания
- Учет бронирований и свободных мест
- Автоматический расчет стоимости проживания с учетом скидок
- Управление дополнительными услугами
- Поиск свободных номеров по различным критериям
- Архивация данных о выбывших постояльцах
- Формирование счетов на оплату
- Программа лояльности для постоянных гостей
